### Python notions_1

## 1. Pip 
* Pip is **control system of package library** written by python. You also can check the package lists of python using pip list command.
<br>(Pip이란, Python으로 작성된 package library를 관리해주는 시스템이다. Pip list를 통하여 컴퓨터에 설치된 python package의 목록을 확인할 수 있다.) 

## 2. Virtualenv
* Virturalenv **provieds the develpoing environment for each python project** for avoiding collision of same library with different versions.
<br>(Virtualenv는 컴퓨터 내에서 독립된 python 개발환경을 만들도록 제공해주는 역할을 한다. 예를 들어 A project에서는 Canmatrix library 1.0version을 사용해야 하고 B project에서는 2.0 version의 Canmatrix library를 사용할 경우 충돌이 발생할 수 있기 때문에 각각의 폴더에 project를 진행할 수 있도록 환경을 제공해주는 것이다.) 

## 3. File type
* There are **.py** which is text file of python source code / **.pyc** which is compiled bytecode file / **.pyo** which is optimized compiled bytecode in Python.
<br>(Python의 file type으로 python source가 들어있는 text file의 .py / compiled bytecode file의 .pyc (.py file이 다른 곳에서 import될 경우 생성된다.) / optimized compiled bytecode file의 .pyo (flag를 주어 실행할 경우 생성된다.) 등이 있다.)
