## Error & Exception 
* While program is working, it will be **worked uncorrectly or finished anomalistically. ** It is **program error** whihc consists **error** and **exception.**
<br>(프로그램이 실행 중 어떤 원인에 의해서 오작동을 하거나 비정상적으로 종료되는 경우를 프로그램 오류라고 하며, 프로그램 오류에는 error(에러)와 exception(예외) 두 가지로 구분할 수 있다.) 

* **Error occurred** if there is **grammar error** during compile, **reference null point** during runtime. These kind of things are able to **cause serious memory or process problem** so it will **be finished anomalistically.**
<br>(에러는 컴파일 시 문법적인 오류, 런타임 시 널포인트 참조와 같은 메모리 또는 프로세스에 심각한 문제를 야기 시켜 프로세스를 종료 실킬 수 있는 오류를 말한다.)

* **Exception occurred** if there is **problem during arithmetic**, **unexpected intteruption** during computer working. These kind of things are able to cause problem in program too but, **it is handled by developer using defensive code.**  
<br>(예외는 연산 도중 넘침, 끼어들기 등 컴퓨터 시스템의 동작 도중 예기치 않았던 이상 상태가 발생하여 수행중인 프로그램이 영향을 받은 오류를 말한다. 하지만, 예외는 개발자가 적절히 코드를 작성해주면 비정상적인 종료를 막을 수 있다.)
