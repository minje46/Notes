# struct

* **`struct`**

  >`struct`는 data를 사용자 정의 자료형으로 define.
  >
  >→ `struct(구조체)` 내부의 variable을 `member` or `field`라고 한다.
  >
  >```c++
  >struct Date
  >{
  >	double year;
  >	int month;
  >    	short day;	
  >    };
  >    ```
  >    
  >    → `struct`의 선언은 `struct`의 구조만 define하고, memory allocate는 되지 않는다.
  >    
  >    → Generally, `struct`의 이름은 Upper case로 설정.
