# Error Message

* **`Runtime Error`**

  > `Runtime Error`는 문법 측면에서 볼 때, code에는 문제가 없으나
  >
  > 잘못된 memory 참조나 논리적 오류가 발생하는 error.
  >
  > * Example
  >
  >   > 1. 잘못된 변수 초기화.
  >   >
  >   >    ```c++
  >   >    Run-Time Check Failure #3 - The variable 'a' is being used without being initialized.
  >   >    ```
  >   >
  >   > 2. Overflow.
